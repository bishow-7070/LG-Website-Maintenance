# To hide official from website if they are not member of municipality

click on content
select officials on type and click filter button to show only officials
select that official or multiple officials and select unpublish selected content and click update button

# To add new officials

hover on content then add content then click on officials
fill the necessary data and check publishing option if published is ticked or not
then click save

# To Chang Range of weight field from default value

hover to structure then content types than officials
click manage fields and search for weight field and click edit
change range value and click save settings

# To add sorting with weight on officials

hover to structure then views than click officials
choose page-staff from displays than at last sort criteria click add button
check for this page (override) is selected
write weight on search and select weight then select sorting asc or desc and click apply setting.
